MTA + COVID-19

Data Visualization

This project explored the impact of the COVID-19 pandemic on New York City MTA subway usage, and whether socioeconomic status had a significant impact on the relative decline of subway usage in different neighborhoods.

The data showed that subway use in lower-income neighborhoods has not decreased as much, which can lead to huge health disparities between people in lower- and higher-income neighborhoods. We produced a series of data visualizations using D3 and data from several sources to show this information in a generally understandable format.

What was used

  • D3
  • HTML/CSS/JS
  • Python
  • NYC Open Data
  • NYT COVID Data
  • New York State Open Data
  • IRS Data
